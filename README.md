# mac-wifi-encrypt-check
リモートワークになり、WiFiの暗号化方式がリスクの1つとなります。

最初は規定でWPA2、WPA3縛りにして、各従業員に自分で暗号化方式を調査してもらおうかと思ったのですが、<br>
従業員にもそれをサポート側も時間がかかるため、スクリプトで取得して脆弱であればウィンドウで警告が出るようにしました。<br>
